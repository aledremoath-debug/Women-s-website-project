# 👗 أناقتي (Anaqati) - متجر أزياء نسائية فاخرة

**أناقتي** هو مشروع واجهة أمامية (Front-end) متكامل لمتجر إلكتروني عصري ومتطور متخصص في الأزياء النسائية الراقية. تم تصميم المشروع ليكون تجربة تسوق فريدة تجمع بين الأناقة، السرعة، وسهولة الاستخدام، مع واجهة مستخدم (UI) احترافية تلبي احتياجات المتاجر العالمية.

---

## ✨ المميزات الرئيسية (Key Features)

- **🎨 تصميم عصري وفخم (Luxury Design):** استخدام تأثيرات الزجاج (Glassmorphism) وتنسيقات لونية متناغمة تمنح الموقع مظهراً احترافياً.
- **🌓 نظام الوضع الليلي والنهاري (Light/Dark Mode):** إمكانية التبديل بين الأوضاع للحصول على أفضل تجربة تصفح.
- **📱 توافق كامل مع جميع الشاشات (Full Responsive):** تصميم "Mobile-First" يضمن عمل الموقع بكفاءة عالية على الهواتف والأجهزة اللوحية والحواسيب.
- **🛒 سلة مشتريات ذكية (Smart Shopping Cart):** إضافة المنتجات، تعديل الكميات، تطبيق أكواد الخصم، وحساب الإجمالي والضرائب والشحن تلقائياً.
- **❤️ قائمة المفضلة (Wishlist):** نظام متكامل لحفظ المنتجات المفضلة والرجوع إليها لاحقاً.
- **🔍 بحث وتصنيف متقدم:** محرك بحث فوري مع تصفية المنتجات حسب (القسم، السعر، اللون، المقاس).
- **💰 دعم العملات واللغات:** نظام مدمج لتغيير العملات (ريال سعودي، دولار، إلخ) ودعم لتغيير حجم الخط.
- **🕒 عروض حصرية:** قسم خاص للعروض الموسمية مع عد تنازلي (Countdown Timer) لزيادة التفاعل.
- **💬 تكامل التواصل المباشر:** زر عائم للتواصل السريع عبر واتساب.

---

## 🛠️ التقنيات المستخدمة (Technologies Used)

- **HTML5:** لهيكلة المحتوى بشكل دلالي (Semantic HTML) محسن لمحركات البحث (SEO).
- **CSS3:** استخدام CSS Variables لتسهيل التخصيص، والرسوم المتحركة المتطورة.
- **Vanilla JavaScript:** كافة الوظائف التفاعلية تم بناؤها بلغة جافا سكريبت الخام بدون مكتبات ثقيلة لضمان السرعة الفائقة.
- **Font Awesome:** لمكتبة الأيقونات التفاعلية.
- **Local Storage:** لحفظ بيانات السلة والمفضلة وإعدادات المستخدم حتى بعد إغلاق المتصفح.

## 🚀 كيفية التشغيل (Quick Start)

المشروع جاهز للعمل فوراً، لا يحتاج إلى تثبيت سيرفر أو مكتبات:

1. قم بتحميل المستودع (Clone the repository):
   ```bash
   git clone https://github.com/your-username/anaqati-store.git
   ```
2. افتح ملف `index.html` في أي متصفح ويب.

---

## 📂 هيكلة الملفات (File Structure)

| الملف | الوصف |
| :--- | :--- |
| `index.html` | الصفحة الرئيسية وهيكل الموقع الكامل |
| `styles.css` | ملف التنسيقات الرئيسي والأنماط التفاعلية والوضع الليلي |
| `app.js` | ملف البرمجة الرئيسي (المنطق البرمجي للسلة، المفضلة، الإعدادات) |
| `data.js` | تحتوي على بيانات المنتجات، الأقسام، وبقية محتوى الموقع |
